🔑 Comment obtenir une clé API Gemini

Guide complet gratuit (5 minutes)

1

📱 Avoir un compte Google

Vous devez avoir un compte Google. Si vous n'en avez pas encore, créez-en un gratuitement :

➜ Créer un compte Google

✓ Si vous avez déjà un compte (Gmail, YouTube, etc.), vous pouvez l'utiliser directement.

2

🌐 Aller sur Google AI Studio

Accédez au site officiel de Google AI Studio :

➜ Aller à Google AI Studio

✓ Cliquez sur le lien ci-dessus
✓ Connectez-vous avec votre compte Google
✓ Acceptez les conditions d'utilisation

3

🔧 Créer une clé API

Une fois connecté, vous verrez un bouton "Create API Key" ou "Obtenir une clé API" :

  1. Cliquez sur le bouton "Create API Key"
  2. Choisissez "Create API key in new project" (première fois)
  3. Attendez quelques secondes pour que la clé soit générée
✅ Une clé API longue s'affichera (comme: AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx)
4

📋 Copier votre clé API

Cherchez le bouton "Copy" à côté de votre clé API :

AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

✓ Cliquez sur le bouton "Copy" pour copier dans le presse-papiers
✓ Ou copiez manuellement (Ctrl+C)

5

🏥 Utiliser votre clé dans MediScan

Retournez à votre application MediScan et :

  1. Ouvrez Test.html ou mobile.html dans votre navigateur
  2. Trouvez le champ "Clé API Google Gemini"
  3. Collez votre clé (Ctrl+V)
  4. Entrez vos symptômes
  5. Cliquez sur "🔍 Analyser avec IA"
✅ Vous recevrez l'analyse médicale alimentée par l'IA Google Gemini !

⚠️ Points importants

🔒 Sécurité: Ne partagez JAMAIS votre clé API avec d'autres personnes. Traitez-la comme un mot de passe !
💰 Gratuit: Google Gemini offre 60 appels gratuits par minute et 1500 appels par jour pour les nouveaux utilisateurs.
✅ Sans carte bancaire: Vous n'avez besoin d'aucune carte bancaire pour les niveaux gratuits.

🆘 Problèmes courants

Q: Je ne vois pas le bouton "Create API Key"

R: Assurez-vous que vous êtes connecté à votre compte Google. Actualisez la page (F5) si rien n'apparaît.

Q: J'obtiens une erreur "Invalid API Key"

R: Vérifiez que vous avez copié toute la clé (elle doit commencer par "AIzaSy"). Évitez les espaces inutiles.

Q: Combien ça coûte?

R: Les 60 premiers appels par minute sont gratuits! Si vous dépassez, il existe des forfaits payants très abordables.
